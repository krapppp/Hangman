# 🎮 Java & MySQL Hangman Game

자바(Java)와 MySQL 데이터베이스를 연동하여 만든 클래식 **행맨(Hangman)** 게임입니다.
DB에서 랜덤으로 단어를 불러와 게임을 진행하며, 사용자의 기록을 저장할 수 있도록 설계되었습니다.

---

## 🛠️ Tech Stack
* **Language:** Java (JDK 17+)
* **Database:** MySQL 8.0
* **IDE:** Eclipse
* **Tool:** DBeaver (Database Management)

---

## 🚀 주요 기능
1. **랜덤 단어 추출:** MySQL DB에 저장된 단어 중 하나를 무작위로 가져와 게임 시작.
2. **카테고리 시스템:** 단어별로 카테고리(과일, 기술, 도구 등)를 제공하여 힌트 부여.
3. **사용자 관리 (진행 중):** 사용자의 점수 및 플레이 기록 저장.

---

## 🕹️ 게임 진행 방법
1. **게임 시작:** DB에서 선택된 단어의 길이만큼 `_` (언더바)가 화면에 표시됩니다.
2. **알파벳 입력:** 단어에 포함될 것 같은 알파벳을 하나씩 입력합니다.
3. **판정:** * 입력한 알파벳이 단어에 있으면 해당 위치의 `_`가 알파벳으로 공개됩니다.
   * 틀린 알파벳을 입력하면 **기회(목숨)**가 차감됩니다.
4. **승리/패배:**
   * 기회가 소진되기 전에 모든 글자를 맞추면 **승리!**
   * 모든 기회를 잃으면 **패배**하며 정답 단어가 공개됩니다.

---

## 📂 데이터베이스 구조
* **words 테이블:** 게임에 사용될 단어와 카테고리 정보 저장.
* **users 테이블:** 사용자 ID, 비밀번호, 최고 점수 저장.
